## 2. Admin — Manual Add / Edit Attendance ✅ (built)
The admin can now manage attendance records directly, not just approve/reject what the mobile app records.

- **Add attendance** button (top-right of the Attendance page) — opens a modal to create a record manually:
  - Choose **Company first, then Project** (project list filters to the chosen company).
  - Choose **Employee**, set **Clock in** / **Clock out** times (leave clock-out blank if still on the clock).
  - Set **Work type** (Site / Factory) and **Status** (defaults to Approved).
- **Edit** (pencil icon on every row) — opens the same modal pre-filled with that record's times, so the admin can **correct the clock-in / clock-out** and save.
- Company is derived from the project on save, so the company/project always stay consistent.
- Times are **timezone-safe** (browser local time → stored as UTC; work date derived from the clock-in day, matching how the mobile app records it).
- Validation: clock-out must be after clock-in; employee + project + clock-in required.
- Styling matches the app (frosted-glass modal, grey-highlight dropdowns, black/grey/white).
