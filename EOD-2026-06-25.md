## 2. Deployment to Netlify ✅ (live)
- Site is deployed: **https://timevera.netlify.app**

## 3. Xero Integration 🔌 (connection working; export in progress)
- Created a new **Xero app** (Web app) with both localhost + production redirect URIs registered.
- Built the full **OAuth connect flow**: "Connect Xero" on each entity → authorise in Xero → tokens stored securely with **auto-refresh** → org attached to the entity.
- ✅ **Connection verified working** — connected **ARKO Joinery**; confirmed token refresh and live Xero API calls succeed.
- Built the **payroll calculation**: turns timesheets into each employee's **hours by pay band** (Ordinary / OT 1.5× / OT 2× / Saturday / Sunday / Public Holiday), using each entity's overtime rules and the public-holiday calendar.
- Added a safe **preview** — verified the maths (e.g. Joshua's 0.5h on a weekday → "Ordinary: 0.5"). Crew never see pay rates: the app sends **hours only**; Xero computes the gross pay.

## 4. Payroll Tab ✅ (now functional)
- Fixed the Payroll page, which was broken (querying fields/statuses that didn't exist).
- **New pay run** now works: it computes and stores each employee's band hours for the period.
- Added **Preview export**, **Approve**, and **Delete** actions per pay run, plus an accurate Xero status banner.
